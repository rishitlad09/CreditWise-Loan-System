# CreditWiseLoanSystem
CreditWiseLoanSystem is a Machine Learning-based loan approval prediction system designed to help financial institutions automate the loan evaluation process.
The model analyzes applicant financial and personal information to predict whether a loan application should be Approved or Rejected.
The goal of this project is to reduce manual effort, minimize human bias, and help banks make faster and more reliable lending decisions.
# Problem Statement
A mid-sized financial company named SecureTrust Bank offers personal and home loans to customers across urban and rural regions of India. Every day, hundreds of customers apply for loans through online platforms and branch applications.

Currently, the bank uses a manual verification process where loan officers evaluate applications by checking income proofs, employment details, credit history, and other documents. This process is time-consuming, inconsistent, and prone to human bias.

Because of this, the bank faces two major challenges:

Good customers sometimes get rejected, resulting in loss of business.

High-risk customers sometimes get approved, leading to financial losses.

To solve this problem, the bank wants to implement an intelligent loan approval system powered by Machine Learning that can automatically analyze applicant details and predict whether a loan should be Approved or Rejected before final human verification.

# Project Objective
The objective of this project is to build a machine learning model that learns patterns from historical loan application data and predicts loan approval decisions accurately.

This system aims to:

Automate loan approval analysis

Reduce human bias in decision making

Improve processing speed

Help financial institutions minimize risk

# Dataset Overview
Each record in the dataset represents a loan applicant and contains features related to financial status, employment, and credit history.

Some important features include:

Applicant Income

Coapplicant Income

Employment Status

Credit Score

Debt-to-Income Ratio

Savings

Loan Amount

Loan Term

Property Area

Education Level

Existing Loans

Target Variable

Loan_Approved

1 → Approved

0 → Rejected
